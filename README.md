Callandt Fusion2 widget configuration for media discovery on Apple TV.

## Import URL

- Recommendations
- Latest shows from: Netflix, Disney+, HBO Max, Apple TV+, Amazon Prime, Paramount+, Hulu
- Movie genre browsing (Action, Adventure, Animation, Comedy, Crime, Documentary, Drama, Fantasy + more)
