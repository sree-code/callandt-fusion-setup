# Callandt Fusion2 Widget Setup

A Fusion2 widget configuration for media discovery on Apple TV.

## What's Included

### Shows
- Trending Shows
- Watchlist: Series
- Recommended Shows
- Latest shows from: Netflix, Disney+, HBO Max, Apple TV+, Amazon Prime, Paramount+, Hulu

### Movies
- Trending Movies
- Watchlist: Movies
- Recommended Movies
- Genre browsing (Action, Adventure, Animation, Comedy, Crime, Documentary, Drama, Fantasy, and more)

## Usage

1. Download `setup-callandt95.json`
2. Import into your Fusion2-compatible app
3. Configure your Trakt credentials
